Two-Module Architecture
🏦
CTS Module
Cheque Truncation System
One AI agent per inward cheque. Solves RBI T+3-hour IET mandate (Jan 2026). 500 cheques → 500 parallel Temporal workflows → entire batch <600ms wall clock. Vision LLM for alteration detection, Siamese network for signature verification, XGBoost for fraud scoring. Missed IET = deemed approval = bank pays regardless.
₹71L cr / year cleared 609M cheques (FY25) 18-month first-mover window IET watchdog at T−30s Qwen2-VL 72B · GOT-OCR2 · XGBoost
🏧
EJ Module
ATM Electronic Journal Intelligence
AI normalisation of Electronic Journal logs across all ATM OEMs. 5+ OEMs (NCR, Diebold, Wincor, Nautilus, GRG), zero standard format — LLM solves this permanently. Dispute resolution with CCTV evidence correlation, fleet observability, predictive maintenance. Cross-sell after CTS foothold.
2.5L+ ATMs deployed 5+ OEMs, zero standard Llama 3.3 70B parser BGE-M3 dispute matching Go edge agent at branch
